# Tic Tac Toe

A simple Tic Tac Toe game built with HTML, CSS, and JavaScript. Two players take turns clicking cells to get three in a row.

## Features

- Two-player gameplay (X vs O)
- Win and draw detection
- Game status display
- Restart button
- Dark theme UI

## Why I Made This

Building a game like Tic Tac Toe is one of the classic token projects everyone tackles at the start of their software engineering journey.
## How to Play

Open `index.html` in a browser, click cells to place your marks, and try to get three in a row. Click "Restart Game" to play again.

## Limitations

- Not production-ready or scalable: this is a small, single-page app with no backend or persistence.
- No AI/bot opponent: only local two-player mode is supported.
- No game modes or difficulty levels.
- No online or networked multiplayer support.
- No persistent score tracking or player profiles.
- Limited accessibility features and mobile refinements.
- No undo/redo or move history.
- Minimal testing and no CI setup.
